Detailed Description

Script that is called by the Camera object after it renders a frame, so that performs PostProcessing before displaying the frame


Parameters
matThe material on which to display the resulting image, a planar screen spanning the Camera's view
mat.shaderThe Shader program bound to the material, shader program that processes pictures displayed onto the material
srcThe color buffer holding the frame after the camera initially renders the scene
destThe buffer in which the post processed image will be stored, what is ultimately displayed onto the screen
player.healthHealth Points of the player, so can apply graphical effects depending on the player's health
player.transform.positionWhere the player is in the world space
cameraReference to the Camera Object, so can extract its projection matrix and situate the player onto the screen
